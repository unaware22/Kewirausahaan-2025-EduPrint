## 📊 2. Perancangan Model Bisnis & Validasi Pasar


### A. Lean Canvas (Model Bisnis)

Strategi bisnis **EduPrint** dirancang untuk menyelesaikan kecemasan mahasiswa terhadap deadline dan kerumitan teknis pencetakan.

#### 1. Masalah & Solusi (Product-Market Fit)
| **PROBLEM (Masalah)** | **SOLUTION (Solusi)** | **UNIQUE VALUE PROPOSITION** |
| :--- | :--- | :--- |
| **1. Akses Jam Kritis:**<br>Tempat print tutup saat malam (22.00+) padahal mahasiswa sering begadang mengerjakan tugas.<br><br>**2. Inefisiensi Waktu:**<br>Antrean panjang di tukang fotokopi konvensional membuang waktu istirahat.<br><br>**3. Kecemasan Format:**<br>Takut salah margin/spasi yang berakibat revisi berulang atau pengurangan nilai. | **1. Layanan 24 Jam + Delivery:**<br>Solusi cetak kapan saja tanpa perlu keluar kosan.<br><br>**2. Platform Website:**<br>Order via Web/WA tanpa perlu install aplikasi (sesuai request responden).<br><br>**3. Jasa "Formatting Check":**<br>Pengecekan margin & layout otomatis sebelum dicetak. | **"Print Kilat 24 Jam: Kirim File via Web, Kami Cek Formatnya, Hasil Diantar ke Depan Pintu."**<br><br>_Solusi cetak dokumen anti-cemas untuk mahasiswa pejuang deadline._ |

#### 2. Eksekusi & Pasar (Market Execution)
| **KEY METRICS** | **UNFAIR ADVANTAGE** | **CHANNELS** |
| :--- | :--- | :--- |
| 1. **Order Malam:** Jumlah transaksi jam 22.00 - 04.00.<br>2. **Zero Error:** Tingkat keberhasilan cetak tanpa revisi.<br>3. **Retention:** % Mahasiswa yang order ulang. | **Quality Control SOP:**<br>Berbeda dengan fotokopi biasa, kami punya SOP pengecekan standar akademik (Margin 4433, Spasi, Page Number) sebelum cetak. | 1. **Website (Landing Page):** Upload file.<br>2. **WhatsApp Business:** Konsultasi cepat.<br>3. **Word of Mouth:** Organisasi kampus. |

| **CUSTOMER SEGMENTS (Target Pasar)** | **EARLY ADOPTERS (Pengguna Awal)** |
| :--- | :--- |
| 1. **Mahasiswa "Deadliners":** Mengerjakan tugas SKS (Sistem Kebut Semalam).<br>2. **Aktivis Organisasi:** Butuh print proposal/LPJ mendadak.<br>3. **Pejuang Skripsi:** Sensitif terhadap kerapian format.<br>4. **Mahasiswa Pekerja:** Tidak punya waktu antre siang hari. | Mahasiswa tingkat akhir (Skripsi) dan Pengurus Harian Organisasi Kampus (HIMA/BEM) yang memiliki mobilitas tinggi. |

#### 3. Keuangan (Financial Viability)
| **COST STRUCTURE (Biaya)** | **REVENUE STREAMS (Pendapatan)** |
| :--- | :--- |
| 1. **Produksi:** Kertas, Tinta, Listrik, Maintenance.<br>2. **Operasional:** Kuota Internet, Hosting Web.<br>3. **Distribusi:** Bensin & Honor Kurir.<br>4. **Marketing:** Iklan Medsos & Promo Awal. | 1. **Jasa Print:** Margin dari harga per lembar.<br>2. **Delivery Fee:** Ongkos kirim flat rate.<br>3. **Priority Charge:** Biaya tambahan untuk layanan "Urgent" atau "Cek Format". |
